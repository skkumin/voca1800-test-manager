# PRD: 영어 단어 시험 자동 생성 및 채점 시스템

## 요약

교사가 선택한 DAY(들)에서 자동으로 5지선다 시험지를 생성하고, PDF 출력 후 채점하는 웹 서비스.
**핵심 가치**: 출제 관리 자동화 + 예문 재사용 방지로 교사의 출제 부담 최소화

---

## 기능

### 1. 시험 생성
**입력**: 
- 모드: DAY 선택 | 미출제 예문 선택
- DAY 선택 시: 1개 이상의 DAY 체크박스
- 문제 수: 입력값

**로직**:
- 대상 단어 조회 (선택된 DAY or 전체)
- 예문별 출제 횟수 조회 (sentence_usage)
- 미출제 예문 우선, 부족 시 가장 오래된 예문 사용
- 각 단어별로 1개 예문 랜덤 선택 → blank 처리
- 오답 4개를 다른 DAY 단어에서 선택
- 보기 5개 랜덤 섞기

**출력**: test_id, 문제 배열 (question/choices/answer)

---

### 2. PDF 시험지
- 이름/반 작성 공간
- 문제 번호 + 예문 + 5지선다 보기
- 인쇄용 레이아웃 (A4)

---

### 3. 교사 입력 채점
- 학생 선택 드롭다운
- 문제별 ①②③④⑤ 버튼으로 답 입력
- 제출 → score + wrong_words 저장

---

### 4. 결과 관리
- 학생별 점수 조회
- 틀린 단어 목록 표시
- (향후) 통계 분석

---

## 기술 스택

| FE | Next.js 14 (App Router) |

## 핵심 제약

✅ **해야 할 것**:
- 예문별 출제 이력 추적 (전역 기준)
- 복수 DAY 동시 선택 가능
- 미출제 예문 우선 출제

❌ **하지 말아야 할 것**:
- OMR 자동 스캔
- 자연어 처리 (lemmatization)
- 사용자 인증
